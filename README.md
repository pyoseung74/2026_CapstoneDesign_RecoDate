# 2026 Capstone Design RecoDate

RecoDate is a capstone project for building an intelligent, place-centered date course recommendation platform.

## Project Summary

- Project name: `RecoDate (Recommend + Date)`
- Goal: recommend optimized date courses around a user-selected anchor place
- Core value:
  - reduce planning time and decision fatigue
  - optimize routes by budget, transport, and time
  - allow interactive editing after recommendation

## Core Features

- anchor-place-centered course generation
- quick start input flow
- deep dive customization
- 3 to 5 full-course recommendations
- open-swap place replacement
- course save and feedback

## Planned Tech Stack

- Frontend: React Native
- Backend: Python FastAPI
- Database: PostgreSQL + PostGIS
- APIs:
  - Kakao Local/Maps API
  - SKT Tmap API
  - Public Data Portal APIs

## Repository Structure

- `app/`: React Native mobile application
- `server/`: Python FastAPI backend
