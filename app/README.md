# App Structure Draft

This directory contains the React Native Expo client.

## Current Scope

The first runnable milestone is a Phase 1 demo:

- main dashboard entry
- initial input screen
- quick start submit
- deep dive optional form
- handoff preview toward Phase 2

## Structure

- `App.tsx`: Expo entry point
- `src/modules/phase1-initial-input`: Phase 1 screens and state
- `src/shared`: domain types, config, and API helpers

The app is still organized by the confirmed product phases:

- `phase1-initial-input`
- `phase2-selection-hub`
- `phase3-customizing`
- `phase4-execution`
